Politique de Confidentialité – MyBot

Dernière mise à jour : 11 Février 2026

MyBot est un chatbot Messenger développé pour répondre automatiquement aux messages des utilisateurs via Facebook Messenger.

1. Données collectées

  • Identifiant public Facebook (PSID)
  • Nom public
  • Messages envoyés au bot
  • Interactions avec le chatbot

2. Utilisation des données

Les données sont utilisées uniquement pour :

  • Répondre aux messages
  • Améliorer l’expérience utilisateur
  • Assurer le fonctionnement du service

3. Partage des données

Nous ne vendons ni ne partageons vos données personnelles avec des tiers.

4. Sécurité

Les données sont stockées de manière sécurisée sur nos serveurs.

5. Suppression des données

Pour demander la suppression de vos données :
